# Privacy Policy — IndeedHarvest

Last updated: July 5, 2026

IndeedHarvest is a browser extension that extracts publicly visible job listing data from Indeed search result pages that you, the user, are actively viewing.

## What We Collect
**Nothing.** IndeedHarvest does not collect, transmit, or store any of your personal data, browsing history, or scraped job data on any external server. All scraped data is stored locally in your browser using Chrome's local storage API (`chrome.storage.local`) and never leaves your device except when you choose to export a file to your own computer.

## Network Requests
The only network request this extension ever makes is a single, user-initiated call to Gumroad's license verification API, triggered only when you manually enter a Pro license key. This call sends only the license key itself (no personal data, no job data) to Gumroad to confirm purchase validity. See Gumroad's own privacy policy for how they handle this request.

## Data You Export
Any CSV, XLSX, or JSON file you export is saved directly to your own device via your browser's normal download mechanism. We never see or receive this file.

## Third-Party Data
The job listing data you scrape originates from Indeed.com and its regional domains. IndeedHarvest is not affiliated with, endorsed by, or connected to Indeed, Inc. in any way. Use of scraped data is subject to Indeed's own Terms of Service, and it is your responsibility as the user to comply with them.

## Changes
We may update this policy if functionality changes. Continued use of the extension after changes constitutes acceptance.
